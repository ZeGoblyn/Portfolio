Furor Fomory (UE 4)

Furor Fomory : First Person Slasher

Projet de validation de Master. Une grande partie du projet était orientée vers la mise en place d'un combat de mêlée qui peut rappeler les jeux Hexen/Heretic. Il comprend deux types d'ennemis et diverses mécaniques communes à ce genre de jeux.


Master degree validation project. An important part of the project was the development of a melee combat system reminding of the Hexen/Heretic games. It contains two types of foes and share common mechanics with this kind of games.



Raid on the Other Side (Unity)

Raid on the Otherside : Survivre aux morts-vivants

Projet destiné à organiser une compétition. Il s'agit d'un jeu de plateforme 2D où le joueur doit survivre cinq minutes contre des vagues d'ennemis dans un niveau dont l'architecture à été inspirée des Metroidvania.


The goal of this project was the organisation of a tournament. It is a 2D platformer game where the player must stay alive for five minutes while facing multiple waves of ennemies. The level Design has been influenced by the Metroidvania genre.



Tutorial Project (UE 4)

Tutorial project : Experimentation

Projet destiné à l'apprentissage de l'Unreal Engine 4. Le but était de créer un niveau de type tutoriel où le joueur peut finir le niveau en recevant le moins d'informations brutes possible.


Project destined to learn how to use Unreal Engine 4. The goal was the creation of a tutorial level where the player have the possibility to reach the end with a minimum of raw information.

The tale of Tara's Fall : Game Design Document

Un Game Design Document portant sur un projet hypothétique. Il contient toutes les rubriques et informations nécessaires à la réalisation du jeu. Celui-ci serait un jeu d'action en vue FPS inspiré de Vermintide et Hexen/Heretic se déroulant dans l'Irlande mythologique.

Level Design Document

Barenton Fountain (UE 4)

Barenton Fountain : Communication de Level Design

Un document de Level Design destiné à un exercice de communication. Le but était de réaliser un document de Level Design complet et de le transmettre à un autre étudiant afin qu'il le réalise. Le projet est un jeu en 2D inspiré de la mythologie Arthurienne. A partir de mon document, il a été porté sur Unity par Jeffrey Zumkir.


A Level Design document destined to a communication exercise. The goal was to create a complete Level Design documentation and giving it to an other student for development. The project is a 2D game inspired by Arthurian mythology. Based on my document, the level was created on Unity by Jeffrey Zumkir.
